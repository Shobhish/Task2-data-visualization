# Task 2: Data Visualization and Storytelling using Power BI

## Objective
Create visualizations that convey a compelling story using charts and graphs.

## Tools Used
- Power BI
- Superstore Dataset

## Task Description
Built an interactive Power BI dashboard using the Superstore dataset to analyze sales, profitability, customer segments, and regional performance.

## Files Included
- Dataset (CSV)
- Power BI Dashboard (.pbix)
- Dashboard Screenshot
- PDF Report

## Dashboard Features
- KPI Cards (Total Sales, Total Orders, Quantity Sold, Total Profit)
- Sales by Category
- Profit by Region
- Sales by Segment
- Sales Trend by Year
- Profit by Sub-Category
- Discount vs Profit Analysis
- Interactive Slicers

## Key Insights
- Total Sales: 2.30M
- Total Orders: 9,994
- Total Quantity Sold: 38K
- Total Profit: 286.40K
- Technology category generated the highest sales
- West region delivered the highest profit
- Consumer segment contributed the largest sales share
- Sales showed steady growth over time

## Outcome
Learned how to use Power BI for data visualization and storytelling by converting raw data into meaningful business insights.
